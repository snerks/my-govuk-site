About

Overview

Are you a well-seasoned open-sourcerer and love the work that the UK Gov Digital Services do?

Are you also like me and have a bit of a thing for the GOV.UK Design System?

The only sane solution would be to turn GDS's production Design System kit into a personal website templating kit.

Tech Stack

This is built using: Astro Components and Layouts, GOV.UK Design System Frontend's JS & CSS, and a lot of tape.

Because of this, not all components from the GOV.UK kit are natively supported.

Thus, you have to manually create a component that can be expanded through Astro Components.
